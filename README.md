# playdate-t9

Very early work in progress keyboard alternative for the Playdate console. Uses a 9-key like input system where the dpad presses the number keys, and the A/B buttons "shift" the dpad target to reach the remaining keys.

Fonts used:

Mono Carlo by tkers: https://tkers.itch.io/mono-carlo

Mono 5x8 by Rob Kohr: https://github.com/RobKohr/robkohr-mono-5x8-font-for-playdate
